# Vim

#### Splitting windows

- **:split** - horizontal split;
- **:vsplit** - vertical split;


#### Buffers

- **:ls** - list buffers;


#### Clipboard

- **gg"+yG** - copy the entire buffer;
- **"+Y** - copy current line;
- **"+y** - copy selected text;
- **"+dd** - cut current line;
- **"+p"** - paste after the cursor;
